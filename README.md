Aplikasi ini merupakan hasil implementasi praktikum Pemrograman Web Lanjut, yang bertujuan untuk menerapkan konsep **Modularisasi** dan **Object-Oriented Programming (OOP)** PHP dalam sebuah aplikasi CRUD sederhana (Create, Read, Update, Delete) data barang.

### Fitur Utama

* **CRUD Data Barang**: Operasi dasar Tambah, Lihat, Ubah, dan Hapus data barang.
* **Arsitektur Modular**: Menggunakan pola *Front Controller* (`index.php`) untuk *routing* dan pemuatan modul secara dinamis.
* **Templating**: Tampilan yang konsisten di semua halaman (Dashboard, Daftar Barang, About) menggunakan `header.php` dan `footer.php`.
* **Database OOP**: Koneksi dan operasi CRUD dikelola melalui **Class `Database`** yang ditulis menggunakan konsep OOP (PHP Class).
